# Elder Care Companion 👵👴

## Features
- Appointment management
- Medication tracking
- SOS emergency alert
- Caregiver dashboard

## Tech Stack
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL

## Setup
1. Clone repo
2. Create database `elderly_carenew`
3. Import SQL file
4. Rename db.sample.php → db.php
5. Add your DB credentials
6. Run on localhost
